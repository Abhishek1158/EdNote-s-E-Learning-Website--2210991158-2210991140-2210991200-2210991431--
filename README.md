# EdNote-s
EdNote's is an e-learning platform that combines multiple learning resources in a single platform like mcq's, smart notes, courses, flashcards and analytics part.

# 🚀🚀 Features
Secure user authentication (Register/Login)
JWT-based session management
Course browsing and lesson viewing
MCQ-based assessments
Smart notes with pin, edit, delete, and search
Flashcards with spaced repetition system
Learning progress tracking
Analytics dashboard for performance insights
# Tech Stack
# Frontend
React.js
React Router DOM
Axios
Tailwind CSS
# Backend
Node.js
Express.js
MongoDB (Mongoose)
JWT Authentication
bcrypt.js
